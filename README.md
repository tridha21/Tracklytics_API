# Tracklytics API

Backend service for Tracklytics – Multi-Guide Project & Evaluation Management System.

## Tech Stack
Spring Boot
Java
MySQL
REST API

## Features
- User signup
- User login
- Get all users
- Update user
- Delete user

## API Endpoints

POST /users/signup  
POST /users/login  
GET /users  
GET /users/{id}  
PUT /users/{id}  
DELETE /users/{id}
